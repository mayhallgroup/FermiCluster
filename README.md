# bc-ci
Block correlated CIPSI under construction

### Installation
1. Download
    
        git clone https://github.com/vibinabraham/bc-ci.git
        cd bc-ci/

2. create virtual environment (optional)
         
        virtualenv -p python3 venv
        source venv/bin/activate

3. Install

        pip install .

4. run tests
    
        pytest test/*.py
